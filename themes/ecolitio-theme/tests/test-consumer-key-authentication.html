WooCommerce Consumer Key Authentication Test

Test Overview

This test validates the new consumer key authentication approach for WooCommerce REST API:

  • 1. Consumer Key Generation - Proper WooCommerce consumer keys generated
  • 2. Authentication Methods - Multiple authentication fallback strategies
  • 3. API URL Construction - Consumer key as URL parameters
  • 4. Permission Handling - Role-based vs consumer key permissions
  • 5. Error Resolution - Should resolve "woocommerce_rest_cannot_create" errors
