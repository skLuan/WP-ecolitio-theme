WooCommerce REST API Nonce Validation Test

Test Overview

This comprehensive test validates:

  • 1. WordPress REST API Nonce Generation - Proper wp_create_nonce('wp_rest') creation
  • 2. Nonce Localization - Nonce correctly passed to JavaScript via wp_localize_script
  • 3. JavaScript Nonce Usage - FormController.js properly retrieves and uses nonce
  • 4. Request Headers - X-WP-Nonce header correctly formatted
  • 5. Error Handling - Proper 403 error detection and user feedback
